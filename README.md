# Innopsi
### G20 2023 theme-based platform to collaborate and solve problems 

### `Develop`

```
npm run develop
# or
yarn develop
```

### `start`

```
npm run start
# or
yarn start
```

### `build`

```
npm run build
# or
yarn build
```
